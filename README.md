# CHATAPP
This an applications were a user can communicate with another user by sending and receiving massages,
user can also add authenticated user in a  friend list or remove user from friend list,
user can send friend request to another user and can accept friend request from another user that is authenticated,
in this application a user will always get notification any time he/she receive massage from another user.

Every authenticated user will have a profile were there contact details can be seen and profile pictures,
user can also login and logout from the app,
user can edit their profile when ever they want to make changes.
